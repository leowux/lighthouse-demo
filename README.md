## Lighthouse 跑分步骤
1. 安装依赖 `npm install`；
2. 运行预览命令 `npm run preview`；
3. 浏览打开 http://localhost:5555 ；
4. 打开 Chrome Devtool 工具，切到 Lighthouse 栏，点击分析页面按钮。
5. 每次修改代码，需要重新运行 `npm run preview` 再进行跑分，修改才会生效。
## 相关命令
```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 打包项目
npm run build

# 生产环境
npm run preview

# 打包分析
npm run analyze
```
